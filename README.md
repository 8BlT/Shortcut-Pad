## Required Components

- Rotary Encoder  
- Arduino Nano 33 IoT  
- M2 and M3 Hex screws  
- Endstop switches  
- 3x7x3mm ball bearings  
- Kailh PG1350 (Choc v1) low-profile keyboard switches  
- Kailh PG1350-compatible low-profile keycaps  

**Note:** Any **PG1350-compatible low-profile keycaps** will work.
